# ACE-Disappear
在游戏运行的时候停止某个软件的进程
# 你需要下载什么
下载最新的release到你的电脑上，并且放到一个你知晓的位置上
# 使用说明

V1.0
启动ACE_Disappear.exe之后，每隔五秒扫描一次进程。
主程序退出方式在托盘里，图标是通知符号
不要拆散主程序和.cmd脚本文件，切勿让它们分开存放
想要开机自启动请管理员身份运行setStartUp.cmd
想要删除开机自启动请管理员身份运行delStartUp.cmd

V0.0版本
进入游戏并且能够正常操作游戏页面之后，切换出游戏窗口，用管理员身份运行”ACEdisappear.cmd“，等待cmd提示成功即可。
如果无法正常打开此脚本，建议以UTF-8模式以文本文件打开，复制命令在cmd中运行
注意：只有进入游戏之后才能使用，游戏启动中使用会使游戏异常报错
# 原理解析
对于某些无法直接结束进程的软件，可以在运行时停止其服务，以此达到结束进程的目的。
